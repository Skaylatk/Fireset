**Fireset** is a game engine written in C, its main goal is to both give that good retro feel and make gamedev easy for everyone.
feel free to use it to develop your game or modify it, also, if you wanto to contribute, just create fork it and create a pull request!
if you find any issue or have a cool idea/suggestion, share it on github issues or send me an e-mail!
